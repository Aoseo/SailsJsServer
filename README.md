# SailsJsServer

 [Sails](http://sailsjs.org) 服务端配置
 
##整合第三方库以及常用库

1. [sails-mongo](https://github.com/balderdashy/sails-mongo)
2. [sails-permissions](https://github.com/tjwebb/sails-permissions)

##功能计划包含

1. 数据库链接配置
2. 应用配置
3. 验证与授权功能
4. 动态数据库功能
5. 后端web界面
6. RESTFul风格API配置
7. 短信验证码功能

##当前已经完成

1. mongodb数据库插件安装
2. permissions验证与权限插件安装
